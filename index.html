Patented Architecture

Three-Level Execution Hierarchy

The only MCP orchestrator with a patented Pipeline → Assembly Line → Radial Burst execution model. Sequential phases. Decision points. Parallel actions.

🔗 Level 1

Pipeline

Sequential phases that represent major milestones. Each phase completes before the next begins. Think: intake → process → deliver.

⚡ Level 2

Assembly Line

Decision moments within each phase. Sequential checkpoints that evaluate conditions, route data, and determine what happens next.

💥 Level 3

Radial Burst

Parallel action execution. Multiple MCP servers fire simultaneously — CRM, email, calendar, database — all in one coordinated burst.

FAQ

Frequently Asked Questions

Everything you need to know about 0nMCP and MCP orchestration.

What is MCP orchestration?
MCP orchestration is the process of combining multiple Model Context Protocol servers into unified, automated workflows. Instead of connecting AI assistants to individual MCP servers one at a time, an orchestrator like 0nMCP federates them under a single gateway — enabling chained, parallel, and conditional execution across services like CRM, email, calendar, and databases through a single trigger.
How do I install 0nMCP?
Run npx 0nmcp in your terminal — that's it. You can also install globally with npm install -g 0nmcp. Add it to your Claude Desktop config and restart. Setup takes about 2 minutes.
Is 0nMCP free?
The core orchestrator is free and open source under the MIT license. The free tier includes 100 API calls/day, full CLI access, and all 252 tools. Pro ($29/mo) and Team ($99/mo) plans unlock higher limits, signed exports, visual builders, and marketplace access.
What services does 0nMCP support?
0nMCP supports 17 services: Stripe, SendGrid, Resend, Twilio, Slack, Discord, OpenAI, Airtable, Notion, GitHub, Linear, Shopify, HubSpot, Supabase, Calendly, Google Calendar, and a full CRM integration with 245 dedicated tools across 12 modules.
How is 0nMCP different from Zapier or Make?
0nMCP uses natural language instead of visual workflow builders. Setup takes 2 minutes vs 30+ minutes per Zapier zap. The core is free and open source vs $20-$100+/month for Zapier. It offers 252 tools with zero maintenance, runs locally on your machine, and uses the patented Pipeline → Assembly Line → Radial Burst execution hierarchy for true multi-server orchestration.
What is the .0n Standard?
The .0n Standard is a universal configuration format for MCP servers — like docker-compose.yml but for AI tools. It standardizes how MCP servers are configured across different clients including Claude Desktop, Cursor, VS Code, Windsurf, and Gemini CLI, solving the incompatible configuration problem.
Can MCP servers work together automatically?
Yes. 0nMCP's patented federation architecture enables MCP servers to work together through automated workflows. A single webhook, API call, or natural language command can initiate multi-phase operations across CRM, email, calendar, database, and other MCP servers simultaneously using the Radial Burst execution pattern.
Is my data secure?
Yes. 0nMCP runs locally on your machine, not in the cloud. API calls go directly to each service — no proxy, no middleman. Credentials are stored locally in ~/.0n/connections/ and never transmitted to third parties. The entire codebase is open source and auditable under the MIT license.

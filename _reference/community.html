Quick Start

Your First Contribution

Get up and running in under 5 minutes.

1

Fork & Clone

Fork the repo on GitHub, then clone it locally.
git clone https://github.com/YOUR_USERNAME/0nMCP.git

2

Install Dependencies

Just one dependency. Minimal by design.
cd 0nMCP && npm install

3

Make Your Change

Add a service to catalog.js, fix a bug, improve docs. Every tool is config-driven — no boilerplate needed.

4

Submit a PR

Push your branch and open a pull request. We review PRs quickly and merge contributions that match the project's quality standards.

Our Values

What We Believe

🌏

Free and Open Source Forever

AI orchestration is infrastructure. It should be accessible to every developer, not locked behind enterprise paywalls or monthly subscriptions. MIT licensed, always.

⚡

Ship Weekly

The codebase is active. We push updates constantly. This isn't a side project — it's infrastructure being built at startup speed by people who use it every day.

🤝

Community Owned

Every unlock milestone is hit by the community, not by a sales team. The more developers who join, the more powerful the platform becomes for everyone.

📈

Config, Not Code

Adding services takes minutes, not hours. Every tool is defined as configuration. Drop it in the catalog and it works. Zero boilerplate, zero ceremony.
